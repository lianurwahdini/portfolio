Recent Work

Here my recent work in Automotive and HVAC (Heating, Ventilasi, Air Conditioning) system. There are six projects (Air Handling Unit, Traction Control System, Stability Control System, Electronic Ignition System, Open Type Compressor, Semi-Hermetic Compressor) done and integrated to Windows and WebGL.

Air Handling Unit (AHU)

AHU is the one of HVAC module. This module have a two mode, Auto - Manual. In this module, user can observe any changes in RH Sensor, CO2 level, Differential Pressure, Mixing Air Temperature, Mixing Air Pressure, Air Outlet Temperature, Air Flow, and Supply Air Pressure.

Traction Control System

TCS is the one of automotive module. In this module, user can control speed using slider as accelerator pedal. User can observe how a car can escape from snowy road and observe braking system in the diagram.

Stability Control System

Stability control system is the one of automotive modules. In this module, user can control the speed using slider as accelerator pedal. Stability control is the braking system, the system will running when understeer, oversteer, and spinning wheels.

Electronic Ignition System

Electronic ignition system is the one of automotive module. In this module, user can control the speed using slider as accelerator pedal. User can observe when the ignite occurs.

Open Type Compressor

Open Type Compressor is the one of HVAC module, explain about operating cycle, suction and dischage gas, piston movement, lubrication, and electronic motor drive. In this module, user can control the speed using slider.

Semi-Hermetic Compressor

Semi-Hermetic Compressor is the one of HVAC module, explain about operating cycle, suction and dischage gas, piston movement, lubrication, oil circulation, motor cooling, refrigerant migration, and electrical connection. In this module, user can control the speed using slider.
